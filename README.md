# MT4599-Simulation-Code

# Optimising Chemotherapy Timing Using Circadian Models

This repository contains the code used for the simulations in my final year project:

"Chronotherapy in Cancer Treatments: Impulsive Modelling and Circadian Optimisation of Chemotherapy"

## Contents

- `simulation.py` – tumour–healthy tissue model
- `optimisation.py` – optimisation of J(theta)
- `plots.py` – figure generation

## Requirements

- numpy
- scipy
- matplotlib

## Usage

Run the scripts to reproduce the figures in Chapter 5 (alter parameters as necessary).
